## Rebuild context

Cowlsly is being rebuilt from a clean slate after prior GitHub repositories and Azure cloud resources were deliberately wiped to reduce confusion and platform drift.

The rebuild principle is: prove small, working pieces first, then build toward the larger long-term vision.

## Repo naming convention

## Active source of truth

`ROADMAP.md` in the repository root is the active build roadmap. Update it when scope changes.
